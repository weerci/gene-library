Обновление базы данных на серевере разработчика.
Необходимо восстановить базу данных, такую же как на пользовательской машине, поэтому все предыдущие исталляции удаяются и база 
устанавливается с нуля. Если были какие разработки, то они должны быть сохранены в виде скриптов, и потом будут накатываться на 
установленную чистую базу.

1. Удаляем текущую инсталляцию:
	drop user modern cascade;
	create user modern identified by "XXXXX";
	grant resource, dba, connect to modern
2. С помощью утилиты экспорта таблиц восстанавливаем таблицы из дампа.
3. С помощью скрипта modern.sql, восстанавливаем исходный код всех процедур, функций и т.д.
4. Устанавливаем права пользователя с помощью скирпта grant.sql, под пользователем sys
5. Компилируем все объекты базы
-------------- База восстановлена-------------------------
6. Накатываем разработческие скрипты для получения новой версии базы.
